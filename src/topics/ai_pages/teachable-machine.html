Teachable Machine

Huấn luyện AI nhận dạng hình ảnh, âm thanh và tư thế người ngay trên trình duyệt — không cần viết một dòng code nào!

1. Teachable Machine là gì?

Teachable Machine là một công cụ trực tuyến miễn phí do Google phát triển, cho phép bất kỳ ai — dù không biết lập trình — có thể tự tay tạo ra và huấn luyện một mô hình Machine Learning (ML) chỉ trong vài phút.

Thay vì phải viết code hay hiểu sâu về toán học, bạn đơn giản chỉ cần: Thu thập ví dụ → Huấn luyện → Xuất mô hình.

2. Nguyên lý hoạt động — Mạng Nơ-ron trong tầm tay

Bên dưới Teachable Machine là một Mạng Nơ-ron Tích chập (CNN) đã được huấn luyện sẵn với hàng triệu hình ảnh (gọi là Transfer Learning). Khi bạn dạy nó nhận diện "Tay giơ lên" hay "Mặt cười", bạn đang thực hiện một quá trình gọi là Fine-tuning.

  • 📸 Image Project: Nhận diện hình ảnh qua webcam hoặc file upload (phân biệt người, vật thể, cử chỉ tay...).
  • 🎵 Audio Project: Nhận diện âm thanh (tiếng vỗ tay, tiếng huýt sáo, tiếng noise...).
  • 🧍 Pose Project: Nhận diện tư thế cơ thể người (ngồi thẳng, cúi đầu, giơ tay...).

3. Thực hành: Huấn luyện AI nhận diện tư thế

Bước 1 — Chuẩn bị Classes: Truy cập teachablemachine.withgoogle.com → chọn Pose Project → Đăng ký ít nhất 2 nhãn (VD: "Đúng tư thế" và "Sai tư thế")

Bước 2 — Thu thập dữ liệu: Cho dataset tối thiểu 30-50 ảnh/class để model đủ "thấu hiểu" đặc điểm. Càng nhiều ảnh đa dạng góc độ, ánh sáng thì model càng chính xác.

Bước 3 — Huấn luyện: Nhấn "Train Model". Teachable Machine sẽ tự động chạy vài chục epochs huấn luyện. Toàn bộ quá trình diễn ra trên trình duyệt của bạn, không cần server.

Bước 4 — Kiểm tra: Dùng webcam Test ngay trên trang. Nếu không hài lòng, thêm data vào các class còn yếu và train lại.

Bước 5 — Xuất và tích hợp: Nhấn Export Model → TensorFlow.js để lấy link nhúng vào web, hoặc TensorFlow Lite để deploy trên thiết bị di động.

4. Ứng dụng thực tế

  • 🏃 Fitness app: Đếm số lần push-up, squat qua camera.
  • ♿ Hỗ trợ người dùng: Ra lệnh cho máy tính bằng cử chỉ tay.
  • 🌱 Nông nghiệp: Nhận diện sâu bệnh trên cây trồng qua ảnh chụp.
  • 🎮 Game: Điều khiển nhân vật game bằng tư thế cơ thể.
🤯 Fun Fact: Toàn bộ quá trình huấn luyện AI mà bạn làm trên Teachable Machine, nếu thực hiện từ đầu bằng code Python, sẽ cần ít nhất 500+ dòng code và kiến thức Deep Learning chuyên sâu.
